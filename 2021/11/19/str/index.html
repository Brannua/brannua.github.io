字符串常量

≥ 0 个字符，被双引号引起来，就是字符串常量（eg：””、”hello”、…）

eg："asd" => 字符串常量 => 会被转换成{'a', 's', 'd', '\0'} => 系统只会保留一份 => 放在一片只读的存储空间中 => 共享

试想：

如果字符串常量所存放的位置不是只读的

那当有很多指针指向它，其中任何一个指针对其的修改，都会导致其它指针对这种修改的不知情

从而可能导致不可预知的严重后果
