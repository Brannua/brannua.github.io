ManjaroLinux 入门指南（一）

2021-09-09- 2021-09-25
2.6k- 9m
Manjaro_tutorial

前言

本教程旨在：

  1. 整理笔者工作流的搭建过程，以防不时之需
  2. 为想要尝试 linux 尤其是 arch 系发行版的小伙伴提供方向
  3. 传递一点 GNU 的 share 精神，给 linux 的社区贡献一点点力量

相比于 windows，我能深刻感受到 linux 带给我的好处：

  1. linux 对电脑本身的硬件配置几乎没有要求，一颗 1Ghz 的 CPU，4GB 内存，128GB 的 SSD 固态硬盘已经满足大多数人的要求（实践完本教程磁盘共使用 13GB 左右）
  2. linux 是开源免费的，这意味着你可以轻松获取其正版系统并查看它的所有源代码，并且你可以在其源代码的基础上，随意修改
  3. linux 拥有十分完善的文档和活跃的社区，你可以轻松查询并处理你所遇到的所有问题（非大佬，能遇到的问题早就有人遇到过了）
  4. linux 上安装软件极其方便，往往一两行命令就能很好地将软件下载并安装，且没有任何捆绑软件
  5. linux 不会收集你的任何个人信息，你对你的系统拥有着绝对掌控权
  6. linux 不存在软件病毒
  7. 你能学到很多有用的知识
  8. ……

个人观点：

虽然 QQ 微信 TIM office 等软件有很多方法在 linux 上运行，但本教程不介绍，因为如果没有特别省心省事的成熟解决方案，那么一个软件在哪种操作系统上的官方支持最好，则让它在哪种操作系统上运行，富贵小白首选购买多台电脑，平民小白首选虚拟机，有极客精神的技术小白可考虑云主机， Wine 兼容层或当今较火的容器技术；因此，本教程不适合“游戏玩家”实践，想要打游戏，最省事的方法就是使用 windows

  • 还在纠结使用哪种 linux 发行版吗？适合自己的才是最好的：Distrochooser
  • 本教程选用的 Linux 发行版：ManjaroLinux
  • 本教程选择的桌面环境：KDE
  • 当然，你也可以尝试使用窗口管理器来代替桌面环境，推荐：dwm
  • 无论你遇到任何关于 ManjaroLinux 的问题，都可以通过以下途径寻找解决方案
    • ManjaroWiki
    • ManjaroForum
    • ArchWiki
    • ArchForum
    • Github
    • Google
    • Baidu
    • 问我
      • 不推荐，除非你真的有认真阅读官方文档并认真思考认真实践
      • 因为学习 linux 本身就需要一定的问题处理能力，其实任何事情都是，自力更生往往比伸手党更能茁壮成长

哦天，你是在想如何安装 ManjaroLinux 吗？这是考验你解决问题的第一个尝试

不过别担心，这位大神=>百度已经提供了各种各样详细而成熟的安装教程，相信天赋异禀的你一定可以轻松搞定

根据个人习惯，调整外设的默认参数

这里是我自己的使用习惯

  • 系统设置–>硬件–>输入设备
    • 触摸板–>勾选反向滚动–>指针加速0.4–>应用
    • 鼠标–>指针速度增加两格–>应用

终端

俗称命令行，即传说中黑客们使用的黑窗口，我们可以通过终端与操作系统进行交互，这也是类 unix 操作系统们最强大的工具

  • 在 ManjaroKDE 中，有如下 3 种方式打开终端，系统中的快捷键均可自定义
    • Konsole => ctrl+alt+t
    • Yakuake => f12
    • 文件资源管理器的内嵌终端 => f4

如果你会接触到软件开发，尤其是 web 前端开发，那么你肯定会经常和 chrome 浏览器的 console 打交道，而一般打开 console 的快捷方式是 f12，这就和 yakuake 的默认快捷键冲突，所以我们需要修改 yakuake 的默认快捷键

  • 系统设置–>工作区–>快捷键–>应用程序–>yakuake–>关掉默认快捷键f12–>设置自定义快捷键为 meta+space

linux 的终端默认以 bash 作为命令处理器，为了获得更好的使用体验，我们可以将 bash 更换为 zsh，只需在终端执行一条命令

chsh -s /bin/zsh

软件包管理器 pacman

顾名思义，你可以使用软件包管理器来下载，安装，卸载任何你想要的软件

但由于国内众所周知的 GFW，若不经合理的设置，我们无法下载到境外的一些软件资源，这会给我们的日常使用造成困扰

因此我们第一步就需要修改软件包管理器背后依靠的软件源，只需连接网络，然后在终端执行一条命令

sudo pacman-mirrors -m rank -c China

由于 ManjaroLinux 官方采用滚动更新的方式更新系统，即官方每隔很短的时间比如几天就会发布一定量的系统更新，但官方还会在某个时间节点将该时间节点最新的系统以 iso 镜像的方式发布到其官网，以此避免用户安装系统后需要再安装大量更新，所以我们刚安装好的“最新”系统其实往往不是最新的，因为有可能官方发布了一些系统更新，但还没有更新官网上的 iso 系统镜像，因此，我们在调整好软件包管理器的软件源后，就需要手动更新系统

同时，推荐所有 Arch 系用户养成时不时尝试接收系统更新并合理备份自己资料的习惯，防止由于一次性接收大量更新而可能造成的混乱

更新系统只需在终端中执行下面一条命令

sudo pacman -Syyu

系统更新完毕之后，手动重启电脑（不建议强制关机或直接拔掉电源，若问缘由，自行回忆或查阅《操作系统》相关的知识）

开机后

  • 你有一定的几率会遇到要求你输入钱包密码的问题，第一次遇到该问题我也很蒙，后来查阅资料知可以按照如下方式解决

    • 依次打开：系统设置–>个性化–>kde钱包
    • 首先，需要启用 kde 钱包子系统，否则无法正常使用记录的账号密码信息
    • 然后，打开右下角的调用钱包管理器，更改密码，设置为空密码即可
    • 重启系统，可以看到密码认证已经跳过

  • 接下来我们尝试使用 pacman 来安装第一个软件，screenfetch，该软件可以在终端中输出你的电脑配置性能的简要信息

sudo pacman -S screenfetch
  • 解释：
    • sudo，可理解为 super user do，在一行命令的开头写上 sudo 就意味着本行命令会以系统管理员的身份执行（会被提示要求输入密码）
    • pacman，软件包管理器
    • -S，安装
    • screenfetch，软件包的名称
  • 安装完毕如何使用 screenfetch，只需在终端执行一条命令
screenfetch
  • 然后我们下载安装一些必要的工具集
sudo pacman -S base-devel fakeroot patch

KRunner

KDE 自带的 KRunner 是一个非常实用的小软件，通过默认快捷键 alt+space 可以在屏幕顶部很方便地将其打开，你可以利用它进行诸如（文件搜索，单位转换，搜索系统中的软件并打开，计算器）等实用的功能

关于语言的调整

默认情况下，用户家目录 ~/ 下会存在一些中文名称的目录，比如“桌面”，“文档”，“下载”，“音乐”，“图片”，“视频”等

由于系统默认没有中文输入法，也为了方便我们在终端中的操作，因此最好将这些中文目录名统一改为英语，一个好的习惯是任何文件（包括目录）的命名都使用英文半角，这在大多数操作系统中都能规避很多莫名其妙的软件错误

当然，将中文目录名修改为英文，我们借助软件包 xdg-user-dirs-gtk

sudo pacman -S xdg-user-dirs-gtk

export LANG=en_US

xdg-user-dirs-gtk-update
  • 此时，系统就会弹出窗口，提示将要把目录名称更新为英文，点击确定即可，然后我们再执行如下命令将系统语言环境改回中文和 UTF-8 字符集
export LANG=zh_CN.UTF-8
  • 然后，系统自带的文件资源管理器 Dolphin 侧栏的位置的子节点映射的路径有可能不正确，需逐个右键点击编辑校对一下
  • 最后，我们执行如下命令删除家目录 ~/ 下的“桌面”目录，因为和“Desktop”目录重复
rm -rf ~/桌面

vim，终端下的文件编辑器

linux 中一切皆文件，为了方便我们在终端中对文件进行编辑，我们需要选用一款在终端中使用的编辑器，比如 nano，vim；本教程均使用 vim

sudo pacman -S vim

配置中文输入法

五十六个民族五十六枝花，我们大家都用中文输入法

# 安装输入法框架
sudo pacman -S fcitx5-im

# 安装输入法引擎和皮肤
sudo pacman -S fcitx5-chinese-addons fcitx5-material-color

# 安装依赖和配置工具
sudo pacman -S fcitx5-qt fcitx5-gtk fcitx5-configtool

# 使用 vim 打开配置文件
vim ~/.pam_environment

# 写入以下配置
GTK_IM_MODULE DEFAULT=fcitx5
QT_IM_MODULE  DEFAULT=fcitx5
XMODIFIERS    DEFAULT=\@im=fcitx5
SDL_IM_MODULE DEFAULT=fcitx5

# 保存退出

  • 重启系统，一般就会自动提示是否启用云拼音，为了获得更好的输入体验，将其勾选

  • 切换中英文输入法的快捷键有两种：ctrl+space 或 ctrl+shift

  • 然后鼠标右键系统托盘键盘指示器，依次点击 “配置”，“拼音”，“配置”

# 我的习惯
1. 勾选启用云拼音（若没有勾选）
2. 勾选在程序中显示预编辑文本
3. 勾选在预编辑中显示完整拼音
4. 点击“应用”！

注意

本教程不提供科学上网教程，但为了能够下载到必须的代码资源，必须解决科学上网这个问题，才能继续阅读并顺利实践本教程！
