要理解上面这段程序背后发生了什么，我们必须了解一些关于 Shell 和其中的缓冲区的知识

Shell

键盘/显示器 <= Shell => 应用程序

Shell 是一个特殊的程序，它能将你的程序 run 起来，也能将运行着的程序 kill 掉

当 Shell 监听到用户按下了 ^C，运行着的程序就被 kill 掉了…
