什么是“链接” ？

记得 2008 年刚接触 windows 的时候，我“卸载”软件都是这么干的 => 将桌面上软件的图标移动到回收站，清空回收站，然后我就觉得软件已经被我卸载了

现在看来，当时的我只是删除了 windows 上应用软件的一个快捷方式而已，并没有将软件真正从我的电脑上卸载

而那时被我删掉的快捷方式就是“链接”的一种实现，显然，“链接”这种东西早就被应用在了计算机中，在我们的生活中扮演着不可或缺的角色

所以，你可以将“链接”狭义地理解成 windows 上的快捷方式，但接下来对“链接”的用途做更多的阐述

hexo + githubActions 搭建的个人博客

2021-09-20- 2021-09-25
1.1k- 4m

GitHub，Gitee，Coding，静态博客「Pages」服务横评

本博客实现的工作流：

  1. 在本地某磁盘某文件夹下编写 .md 博客文件

  2. 以 ssh 的方式将博客站点的存储库拉取到本地，将写好的 .md 源文件拷贝到本地博客站点存储库中

  3. 以 git 的方式形成提交，push 到远端，自动触发远端博客站点存储库的 githubActions

  4. githubActions 自动对博客源码进行编译部署

注意：虽然本博客支持云端写作触发自动化部署，但是不推荐这么做，因为这样会导致在本地某磁盘某文件夹下编写的 .md 博客文件滞后于远端博客站点存储库

rss 新手入门指南

2021-09-12- 2021-09-25
1.5k- 5m

本文基于这篇文章，对其进行了精简和提炼并补充记录了笔者实践 rss 的步骤，在此感谢这篇带我入门 rss 的文章的作者，致敬让我们的生活更方便的每一位开发者

为什么会诞生 rss 这么个东西（有需求，才会有产出）

  • 生活中，你获取信息的方式可能是这样的
    • 比如你想获取一个歌手的最新消息，可能就需要关注他的（微博、B站、抖音、……），于是你就需要下载安装这些平时你可能并不使用的 APP，并在订阅关注该歌手的同时给这些新安装的 APP 打开通知权限
    • 于是这些 APP 就会频繁地往你手机下拉通知面板中发送各种消息，如果你没给手机设置静音或免打扰模式，你的工作就很可能被这些时不时主动发送到你手机上的消息打断甚至打乱
    • 当你有一会儿没看你手机的下拉通知面板，里面就很可能接收到了一大堆消息，这些消息不仅杂乱无章，并且很可能含有很多由推荐算法和商业广告发来的垃圾消息，导致你很难快速地定位到你想知道的消息
    • 甚至你很可能一不小心就点到了你手机下拉通知面板的清空按钮，造成你可能就错过了你想知道的消息
  • 总结
    1. 你不想安装一大堆你并不使用的 APP
    2. 你需要这样一个入口
      1. 这个入口中拥有所有你想要及时获取的消息，而没有（各种推荐算法，商业广告）发来的垃圾消息
      2. 这个入口中的消息可以被很好地归类，而不是像手机下拉通知面板中那样消息杂乱无章
      3. 这个入口中的消息可以被很好地保存下来，但每条消息本身都有“已读”和“未读”两种状态
      4. 这个入口可以在任何时候被你主动打开查阅，而不是像手机下拉通知面板中那样主动打扰你的手机

ManjaroLinux 入门指南（一）

2021-09-09- 2021-09-25
2.6k- 9m

前言

本教程旨在：

  1. 整理笔者工作流的搭建过程，以防不时之需
  2. 为想要尝试 linux 尤其是 arch 系发行版的小伙伴提供方向
  3. 传递一点 GNU 的 share 精神，给 linux 的社区贡献一点点力量

相比于 windows，我能深刻感受到 linux 带给我的好处：

  1. linux 对电脑本身的硬件配置几乎没有要求，一颗 1Ghz 的 CPU，4GB 内存，128GB 的 SSD 固态硬盘已经满足大多数人的要求（实践完本教程磁盘共使用 13GB 左右）
  2. linux 是开源免费的，这意味着你可以轻松获取其正版系统并查看它的所有源代码，并且你可以在其源代码的基础上，随意修改
  3. linux 拥有十分完善的文档和活跃的社区，你可以轻松查询并处理你所遇到的所有问题（非大佬，能遇到的问题早就有人遇到过了）
  4. linux 上安装软件极其方便，往往一两行命令就能很好地将软件下载并安装，且没有任何捆绑软件
  5. linux 不会收集你的任何个人信息，你对你的系统拥有着绝对掌控权
  6. linux 不存在软件病毒
  7. 你能学到很多有用的知识
  8. ……

个人观点：

虽然 QQ 微信 TIM office 等软件有很多方法在 linux 上运行，但本教程不介绍，因为如果没有特别省心省事的成熟解决方案，那么一个软件在哪种操作系统上的官方支持最好，则让它在哪种操作系统上运行，富贵小白首选购买多台电脑，平民小白首选虚拟机，有极客精神的技术小白可考虑云主机， Wine 兼容层或当今较火的容器技术；因此，本教程不适合“游戏玩家”实践，想要打游戏，最省事的方法就是使用 windows
